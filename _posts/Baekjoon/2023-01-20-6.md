## Solution

```python
N, X = map(int, input().split())
A = list(map(int, input().split()))

for i in range(N):
    if X > A[i]:
        print("%d" %A[i], end = ' ')
```        
- A: input List
- N: len(A), 리스트 A의 요소의 개수
- X: 비교를 위한 기준
